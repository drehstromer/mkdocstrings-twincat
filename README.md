A Twincat handler for mkdocstrings.

## Installation

With `pip`:
```bash
pip install mkdocstrings-twincat
```

With [`pipx`](https://github.com/pipxproject/pipx):
```bash
python3.8 -m pip install --user pipx
pipx install mkdocstrings-twincat
```
